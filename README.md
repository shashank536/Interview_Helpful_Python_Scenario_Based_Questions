## 🚀 How to Run

Clone the repository:

```bash
git clone https://github.com/shashank536/Interview_Helpful_Python_Scenario_Based_Questions.git
```

Navigate to the desired folder and run any solution:

```bash
# For Python Basic folder
cd "Python Basic"
python practice_scenario_based.py

# For Python Programming Basic Assignment folder
cd "Python Programming Basic Assignment"
python Q01.py
```
